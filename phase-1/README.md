# Phase 1 (ICS 433)

This folder holds **course Phase 1** material for the Memory Allocation Simulator project: what you submit, export, or archive for grading, without mixing it into application source under `src/`.

## Contents

| Subfolder | Use for |
|-----------|---------|
| **`deliverables/`** | Reports, screenshots, exported PDFs or zips you turn in, milestone write-ups. |
| **`references/`** | Optional copies of the assignment PDF, rubric, or notes (avoid secrets and personal data). |

Large binaries (PDFs, images) can bloat the repo; prefer Git LFS or keep heavy files local and list paths in your report.
